# Hospital Resource Management and Coordination System (HRMCS)

A full-stack capstone MVP for coordinating hospital resources across multiple facilities.

## Features
- Multi-hospital dashboard
- Inventory and shortage tracking
- Resource request workflow
- Approval flow for transfer requests
- AI-style recommendation engine for best matching hospitals

## Run the app

### Backend
```bash
cd backend
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm start
```

Open the frontend at http://localhost:3000 and the backend health endpoint at http://localhost:4000/api/health.
